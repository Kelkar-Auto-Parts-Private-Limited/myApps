Admin / Super Admin Training Guide

Role: Super Admin / Admin

This guide covers all administrative functions in the KAP myApps Portal. As an Admin, you have access to user management, all applications (VMS, HRMS, HWMS, Security), database storage monitoring, and master data configuration.

Table of Contents
1. Logging In 2. Portal Overview 3. User Management 4. App Access 5. DB Storage View 6. Master Data Management 7. Profile & Settings

1. Logging In

You see the KAP login page with the company logo, username/password fields, a math CAPTCHA, and a "Remember me" checkbox. A connection status indicator shows database connectivity at the bottom.
Step 1: Enter your username and password.
Step 2: Solve the math CAPTCHA (e.g., "7 + 3 = ?").
Step 3: Click Sign In. If credentials are correct, you enter the portal.
After 5 failed attempts, the account locks for 60 seconds. Wait for the timer to expire.
If prompted to change your password on first login, set one with 6-12 characters including uppercase, lowercase, number, and special character.

2. Portal Overview

The portal shows a left sidebar with: Apps, Users, Profile, and DB Storage links. The main area displays the Apps grid with tiles for VMS, HRMS, HWMS, and Security. Your name and role appear in the sidebar header.

Click any app tile to open it. The sidebar highlights your current section. A connection status widget at the bottom shows your database link status (green = connected).

3. User Management

Viewing Users

Step 1: Click Users in the sidebar. You see a table with columns: Photo, Full Name, Username, Location, Apps, Roles, and Actions.
Step 2: Use the search bar to filter by name. Toggle "Show Inactive" to include deactivated accounts.

Adding a New User

Step 1: Click + Add User. A modal opens with fields for Username, Password, Full Name, Location, and Mobile.
Step 2: Under Apps Access, check which apps (VMS, HRMS, HWMS, Security) the user needs.
Step 3: For each enabled app, assign roles. VMS roles include: Trip Booking, KAP Security, Material Receiver, Trip Approver, Plant Head, Vendor. HWMS and HRMS have their own role sets.
Step 4: Click Save.
Mark a user as Inactive (red checkbox) instead of deleting them to preserve audit history.

Editing / Resetting Passwords

Step 1: In the Users table, click the edit icon on the user's row.
Step 2: Modify any field. To reset the password, type a new value in the Password field.
Step 3: Click Save to apply changes.

4. App Access

As Admin, you can open all four apps from the portal grid:

  • VMS — Vehicle Management System: trip booking, security gate ops, approvals, master data
  • HRMS — HR Management: employees, attendance, salary, masters
  • HWMS — Warehouse Management: containers, invoices, material requests, payments
  • Security — Surveillance: checkpoints, guards, round schedules
Each app opens in the same browser tab. Use the "My Apps" button in any app's sidebar to return to the portal.

5. DB Storage View

Step 1: Click DB Storage in the portal sidebar.
A "Database Storage Analysis" panel appears showing table-by-table row counts and estimated sizes. A Refresh button reloads the data.
Step 2: Click Refresh to get the latest storage statistics.
Use this view to monitor database growth and identify large tables that may need cleanup.

6. Master Data Management

In VMS, the Admin has access to these master tables (accessible from the VMS sidebar):

Locations

Navigate to Location Master. Add or edit locations (KAP plants, external sites). Each location can have assigned Trip Booking users, Material Receivers, Trip Approvers, and a Plant Head.

Vehicle Types, Vehicles, Drivers, Vendors

Each master has a dedicated page with a sortable table, Add/Edit/Delete actions, and Export/Import buttons for bulk operations via Excel (.xlsx).
Use the "Hide Inactive" toggle to keep the table clean. Export before making bulk changes as a backup.

Trip Rates

Navigate to Trip Rate Master. Define rates per vehicle type + route combination with effective date periods. Use the "Suggest Trips" button to find routes from booked trips that lack rate definitions. Rates can require approval before becoming active.

7. Profile & Settings

Step 1: Click Profile in the sidebar to view/edit your full name, mobile, email, and profile photo.
Step 2: Expand Change Password to update your credentials.
