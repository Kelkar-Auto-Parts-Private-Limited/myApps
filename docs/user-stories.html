Portal

Authentication, navigation, user management, and profile settings
Authentication
PRT-01
User
As a user, I want to sign in with my username and password (with math CAPTCHA verification) so that only authorized personnel can access the platform.
PRT-02
User
As a user, I want to choose "Remember me" to stay logged in on my device so that I do not have to re-enter credentials each time.
PRT-03
User
As a user, I want the system to lock my account after too many failed login attempts so that brute-force attacks are prevented.
Profile & Password
PRT-04
User
As a user, I want to view and edit my profile (name, mobile, email, photo) so that my personal information stays current.
PRT-05
User
As a user, I want to change my password from the profile screen so that I can maintain account security.
App Navigation
PRT-06
User
As a user, I want to see a grid of applications I have access to (VMS, HRMS, HWMS, Security) so that I can navigate to the relevant app in one click.
PRT-07
User
As a user, I want the portal to show a real-time database connection status indicator so that I know whether the system is online before I start working.
User Management (Admin)
PRT-08
Admin
As an admin, I want to add, edit, and deactivate user accounts with role and location assignments so that access is tightly controlled across all apps.
PRT-09
Admin
As an admin, I want to reset a user's password so that I can help employees who are locked out.
DB Storage
PRT-10
Super Admin
As a super admin, I want to view a database storage analysis (row counts, image sizes, egress estimates) so that I can monitor usage and plan capacity.

VMS -- Vehicle Management System

Trip booking, gate operations, material receipt, approvals, and fleet masters
Dashboard & Reports
VMS-01
Admin
As an admin, I want to view an operations dashboard with overview stats, trip details, and reports filtered by date range so that I have a real-time picture of all vehicle activity.
VMS-02
Admin
As an admin, I want to generate and download trip reports as Excel or PDF so that I can share operational data with management.
Trip Booking
VMS-03
Booking User
As a trip booking user, I want to create a trip with multi-segment, multi-destination routing (selecting vendor, vehicle type, driver, and vehicle) so that complex logistics are captured in a single booking.
VMS-04
Booking User
As a trip booking user, I want to advance each trip segment through its lifecycle (booked, gate out, destination, gate in, completed) so that real-time trip status is always visible.
VMS-05
Booking User
As a trip booking user, I want to capture photos at each segment step (e.g., gate entry/exit) so that there is visual proof of operations for audit purposes.
Gate Operations (KAP Security)
VMS-06
KAP Security
As a KAP Security guard, I want to record vehicle gate-in and gate-out with timestamp and photo so that every vehicle movement through KAP premises is tracked.
VMS-07
KAP Security
As a KAP Security guard, I want to log a spot vehicle entry for unscheduled vehicles so that ad-hoc arrivals are recorded even without a pre-booked trip.
Material Receipt
VMS-08
Material Receiver
As a material receiver, I want to confirm receipt of goods for incoming trip segments so that the trip can progress to completion and delivery is verified.
Trip Approvals
VMS-09
Trip Approver
As a trip approver, I want to review and approve or reject completed trips so that only verified trips proceed to billing.
Vendor View
VMS-10
Vendor
As a transport vendor, I want to view my assigned trips and their current status so that I can coordinate my fleet and drivers accordingly.
Fleet & Rate Masters
VMS-11
Admin
As an admin, I want to manage a location master (KAP plants and external destinations) so that trip routing options are always up to date.
VMS-12
Admin
As an admin, I want to manage vehicle types, vehicles (with registration and insurance expiry), and drivers so that the fleet roster is accurate for bookings.
VMS-13
Admin
As an admin, I want to manage vendors (transport companies) so that booking users can assign trips to approved carriers.
VMS-14
Admin
As an admin, I want to define trip rate rules (per route, vehicle type, and vendor) with approval workflow so that trip costs are automatically calculated and controlled.
VMS-15
Super Admin
As a super admin, I want to access a system helper utility for data correction and diagnostics so that I can resolve data issues without direct database access.

HRMS -- HR Management System

Employee records, attendance, salary, leave, and organizational masters
Employee Management
HRM-01
HR Manager
As an HR manager, I want to add, edit, and view employee records (personal details, employment type, plant, team, category, designation) so that the workforce database is complete and accurate.
HRM-02
HR Manager
As an HR manager, I want to import employees in bulk from Excel/CSV and export the employee list so that large onboarding batches and audits are efficient.
HRM-03
HR Manager
As an HR manager, I want to bulk-update employee data and import DOJ/DOB dates from a spreadsheet so that corrections can be applied at scale.
HRM-04
HR Manager
As an HR manager, I want to filter employees by plant, type (On Roll / Contract / Piece Rate / Visitor), team, category, and status so that I can quickly find the records I need.
HRM-05
HR Manager
As an HR manager, I want to manage employee change requests (ECR) with approve/reject workflow so that sensitive field changes are reviewed before taking effect.
HRM-06
HR Manager
As an HR manager, I want to record employment periods (joining, separation, rejoin) per employee so that the full employment lifecycle is tracked.
Attendance
HRM-07
HR Manager
As an HR manager, I want to view a monthly attendance grid showing daily entry/exit times, present/absent/half-day status, and overtime so that I can verify workforce presence at a glance.
HRM-08
HR Manager
As an HR manager, I want to view a P & OT (Present and Overtime) summary per employee for the month so that payroll inputs are ready for calculation.
HRM-09
HR Manager
As an HR manager, I want to import and export attendance data via Excel templates so that data from biometric systems can be integrated.
Salary & Statutory Compliance
HRM-10
HR Manager
As an HR manager, I want to calculate monthly salary for workers and staff (with separate views) based on attendance, wage rates, and overtime so that payroll is computed accurately.
HRM-11
HR Manager
As an HR manager, I want to configure salary settings including day types (Working/Holiday/Paid Holiday), OT rates, and PL (Privilege Leave) rules per category so that calculations follow company policy.
HRM-12
HR Manager
As an HR manager, I want the system to automatically apply statutory deductions (PT, PF, ESI) according to government slabs so that compliance is ensured without manual lookup.
Print Formats & Masters
HRM-13
HR Manager
As an HR manager, I want to create and manage salary print formats (filtered by plant, type, category, team) and bulk-print salary slips so that payroll documents are generated efficiently.
HRM-14
HR Manager
As an HR manager, I want to manage organizational masters (Plants, Categories, Employment Types, Teams, Departments, Sub-Departments, Designations) so that the HR structure mirrors the company's organization.

HWMS -- HGAP Warehouse Management System

Import container tracking, invoicing, material requests, payments, and inventory
Dashboard & Overview
HWM-01
Warehouse Manager
As a warehouse manager, I want to view a dashboard summarizing active containers, invoice totals, pending material requests, and payment status so that I have a consolidated operational view.
Container Tracking
HWM-02
Warehouse Manager
As a warehouse manager, I want to track import containers (by sea and by air) through their full lifecycle (booking, dispatch, in-transit, warehouse arrival) so that I know the real-time status of every shipment.
HWM-03
Warehouse Manager
As a warehouse manager, I want to link containers to main invoices and track pickup dates and reach dates so that container movements and their commercial documents stay synchronized.
Invoice Management
HWM-04
Warehouse Manager
As a warehouse manager, I want to create and manage Main Invoices (MI) with line items, amounts, gross weight, and tariff details so that import documentation is complete and auditable.
HWM-05
Warehouse Manager
As a warehouse manager, I want to advance MI status through a workflow (Draft, RFD, Dispatched, Warehouse) so that the team can track each invoice's progress.
HWM-06
Warehouse Manager
As a warehouse manager, I want to create Sub-Invoices (SI) linked to a Main Invoice for partial or split shipments so that billing and inventory are tracked at a granular level.
HWM-07
Warehouse Manager
As a warehouse manager, I want to import and export invoice data from/to Excel so that bulk data entry and external reporting are efficient.
Material Requests & Inventory
HWM-08
Warehouse Manager
As a warehouse manager, I want to create and track material requests against inventory so that parts are dispatched from the warehouse to production in a controlled manner.
HWM-09
Warehouse Manager
As a warehouse manager, I want to view a real-time inventory of received parts with quantities so that I know what is available in the warehouse at any time.
Payments & Outstanding
HWM-10
Accounts Manager
As an accounts manager, I want to record payments against Main Invoices and track received vs. balance amounts so that vendor payables are clear and up to date.
HWM-11
Accounts Manager
As an accounts manager, I want to view an outstanding report showing unpaid balances per invoice so that overdue payments are identified promptly.
Masters
HWM-12
Warehouse Manager
As a warehouse manager, I want to manage a Part Master (part numbers, revisions, descriptions, UOM, net weight, HSN, packing, rates) with import/export so that all part data is centralized and consistent.
HWM-13
Warehouse Manager
As a warehouse manager, I want to manage Customer, Carrier, and Port masters so that invoices and containers can reference validated entities.
HWM-14
Admin
As an admin, I want to configure company details and manage other masters (HSN codes, UOM, packing types) so that system-wide reference data is maintained centrally.

Security Surveillance

Checkpoint, guard, and round schedule management for KAP facilities
Checkpoint Master
SEC-01
Admin
As an admin, I want to define security checkpoints per KAP location (with name, description, and sort order) so that all surveillance points are catalogued.
SEC-02
Admin
As an admin, I want to filter checkpoints by location and activate or deactivate them so that the checkpoint list reflects current facility layout.
Guard Master
SEC-03
Admin
As an admin, I want to add and manage security guards (name, mobile, location, shift, photo) so that the guard roster is always up to date.
SEC-04
Admin
As an admin, I want to assign guards to Day, Night, or General shifts so that shift coverage is properly planned.
SEC-05
Admin
As an admin, I want to toggle guard active/inactive status and filter the list accordingly so that only current personnel appear in scheduling.
Round Schedules
SEC-06
Admin
As an admin, I want to create round schedules that assign a guard to a sequence of checkpoints with a defined frequency and time window so that patrol routes are standardized.
SEC-07
Admin
As an admin, I want to filter round schedules by location and active status so that I can quickly review coverage for a specific facility.
SEC-08
Security Guard
As a security guard, I want to view my assigned round schedules with checkpoint order and time windows so that I know exactly where and when to patrol.
