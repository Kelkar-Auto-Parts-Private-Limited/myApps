📗 Attendance Excel Converter
Upload the biometric workbook. Each sheet represents one date and must have: Column A = Emp Code, Column C = Employee Name, Column D = Date, Column F = Time IN, Column G = Time Out. The first row is treated as a header and skipped. Rows from every sheet are merged into a single table you can review and download.
📊 Daily Attendance Summary
📅 From history:
Two ways to generate the summary:
1. Upload Excel — raw biometric file with Column B = Date & Time, Column C = In/Out, Column D = Emp Code. Only IN records count; each employee's latest IN timestamp is used.
2. Pick a date from history — uses the attendance data already imported into the system for that month. Head count is grouped with departments in rows and plants in columns.
📈 Monthly Headcount Graph

📥 Import / Update ESSL Attendance
Upload ESSL punch data for the selected month. Only one month at a time.
Excel columns: Emp Code · Employee Name (optional) · Date (DD/MM/YYYY or YYYY-MM-DD) · Time IN · Time Out
Notes: All rows must be for the same month · Existing data can be replaced on import · Times can be HH:MM or Excel time decimals.
📥 Import Salary Revision (On Roll)
Upload an Excel with columns: Emp Code · Role · Sal/Day · Sal/Month · Allowance/Month · Effective Month.
Applies only to On-Roll employees. Each approved row creates a new revision effective from the given month; every other period field is copied forward.
📋 Import Alteration
Upload manual corrections to attendance punches (with reason) for the selected month.
Excel columns: Emp Code · Date · Time IN · Time Out · Reason
Notes: All rows must be for the currently selected month · Alterations override ESSL attendance during salary calculation.
Add Alteration Manually Single-row entry — overrides ESSL punch for that employee on that day.

📥 Import Contract Salary Revision
Upload an Excel with columns: Emp Code · Role · Sal/Day · Sal/Month · Allowance/Month · Effective Month.
